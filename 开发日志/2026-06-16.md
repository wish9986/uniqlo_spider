## 关键决策记录

| 决策 | 选项 | 最终选择 | 理由 |
|------|------|----------|------|
| 爬取策略 | Selenium / API | API策略 | 发现Commerce API，数据结构完整 |

| 高级功能 | 去重/导出/代理 | 仅MD5去重 | 满足核心需求 |

## 技术发现

- 优衣库 US 站使用 Commerce API v5
- 分类 ID 22211=男装，95669=衬衫
- 图片CDN：`image.uniqlo.com`
- 图片路径规律：`/{region}/imagesgoods/{productId}/{type}/...`
- SSL 连接经 VPN 代理不稳定，需完善重试机制

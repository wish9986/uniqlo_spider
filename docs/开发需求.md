# 开发需求文档

## 1. 项目概述

- **项目名称**：优衣库国际官网商品图片爬取 (uniqlo_spider)
- **项目类型**：数据爬取工具 (Web Scraper)
- **技术栈**：Python (requests, lxml, Pillow, Selenium)
- **开发周期**：单次迭代（2026-06-16）

## 2. 业务目标

从优衣库国际官网 US 站点的商品列表页中，自动化爬取所有商品的详情页默认颜色轮播主图，保存到本地。

## 3. 功能需求

### 3.1 核心功能

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| F1 | 商品列表获取 | P0 | 从 API 获取男装衬衫分类所有商品 |
| F2 | 商品详情获取 | P0 | 逐个获取商品详情数据 |
| F3 | 默认颜色主图提取 | P0 | 提取默认展示颜色的一张或多张轮播图 |
| F4 | 图片并发下载 | P0 | 多线程下载图片到本地 |
| F5 | 数据源探测 | P0 | check_source.py 探测数据源可行性 |

### 3.2 基础功能（自动集成）

| 编号 | 功能 | 说明 |
|------|------|------|
| B1 | 断点续传 | progress.json 记录进度，中断后可恢复 |
| B2 | 自动翻页 | API offset 分页 |
| B3 | 失败重试 | 3次+指数退避，连续5次失败终止 |
| B4 | 实时进度保存 | 每完成一个商品后更新 |
| B5 | 请求延迟 | 默认 1.5 秒 |
| B6 | 图片URL清洗 | 去掉 width 参数 |
| B7 | 规范命名 | {名称缩写}_{商品编码}_{序号}.jpg |
| B8 | 日志系统 | 控制台 + 文件 + 独立错误日志 |
| B9 | UA 轮换 | 内置 UA 池随机切换 |
| B10 | 图片校验 | 文件大小 + 文件头验证 |
| B11 | MD5 去重 | 避免重复下载 |

### 3.3 可选功能（本次不启用）

- 代理 IP 轮换
- 元数据导出 (CSV/JSON)
- 运行模式切换（快速/完整）
- 自定义图片存储结构

## 4. 数据范围

- **入口页面**：https://www.uniqlo.com/us/en/men/shirts-and-polos
- **目标分类**：男装衬衫（API path: 22211,95669,,）
- **商品数量**：85 个
- **图片范围**：每个商品的默认展示颜色对应的轮播图（主图 + 模特展示图）
- **预计图片数**：约 100~200 张

## 5. 约束条件

- 需通过 VPN（127.0.0.1:7892）访问目标网站
- 需使用虚拟环境隔离依赖
- 开发过程中不提交 Git 仓库
- 项目文件保存在 E:\workpace\ 下
